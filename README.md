# Typing Speed Test 

A clean and interactive web-based typing speed test application that helps users measure and improve their typing skills.

##  Features

- **Multiple Test Durations**: Choose from 30 seconds, 1 minute, or 2 minutes
- **Real-time Feedback**: Live typing speed (WPM) and accuracy tracking during the test
- **Visual Text Highlighting**: 
  - Green highlighting for correct characters
  - Red highlighting with underline for incorrect characters
  - Blinking cursor showing current position
- **Comprehensive Statistics**:
  - Gross WPM (Words Per Minute)
  - Net WPM (accounting for errors)
  - Accuracy percentage based on total keystrokes
  - Total words typed
  - Total keystroke count
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Random Text Generation**: Varied sentences and phrases for each test

##  How to Use

1. **Select Duration**: Choose your preferred test length (30s, 1min, or 2min)
2. **Start Test**: Click the "Start Test" button
3. **Type Away**: Type the displayed text as accurately and quickly as possible
4. **View Results**: After completion or stopping, see your detailed typing statistics

##  Metrics Explained

- **Gross WPM**: Raw typing speed based on correct characters typed
- **Net WPM**: Typing speed after accounting for errors (more accurate measure)
- **Accuracy**: Percentage of correct keystrokes (includes backspaces in calculation)
- **Words Typed**: Total words completed (based on 5 characters per word standard)

Easy to modify:
- Add new sentences in the `sentences` array in `script.js`
- Adjust time durations by modifying the time selection buttons
- Change color scheme by updating CSS variables
- Add new features like difficulty levels or custom text input

## Tips for Better Typing

- Focus on accuracy first, speed will follow
- Use proper finger positioning (touch typing)
- Practice regularly for consistent improvement
- Don't look at the keyboard while typing
- Take breaks to avoid fatigue
